🚀 BrowseMind — Intelligent Price Tracking Platform
✨ Features
🔍 Real-Time Price Monitoring

Continuously fetches the latest product prices and availability from leading platforms such as Amazon and Flipkart.

📈 Price History & Predictive Insights

Analyze historical price trends and leverage AI-driven forecasts to anticipate future price drops during major sales events.

💎 Modern UI/UX Experience

Designed with a futuristic approach using:

Glassmorphism design system
Smooth animations and transitions
Interactive 3D elements
📊 Deal Intelligence System

Evaluate every purchase with:

Deal Score — measures value for money
Trust Score — assesses seller reliability
🔔 Smart Notifications

Receive real-time alerts when a product reaches your desired price point.
🛠️ Tech Stack
| Layer          | Technologies                                                    |
| -------------- | --------------------------------------------------------------- |
| **Frontend**   | React 18, Vite, Framer Motion, Chart.js, Three.js, Lucide React |
| **Backend**    | Node.js, Express, Axios, Cheerio                                |
| **Styling**    | Custom CSS (Glassmorphism Design System)                        |
| **Deployment** | Vercel                                                          |

Frontend → http://localhost:5173
Backend API → http://localhost:5000

📁 Project Structure
